# Japan Earthquake Fuzzy Logic Project

Main files:

- `Earthquake_Fuzzy_Logic_Project.ipynb` - notebook for submission.
- `earthquake_fuzzy_logic.py` - simple manual input-output fuzzy program.
- `Japan_earthquakes_2001_2018_cleaned.csv` - cleaned dataset used as project data.
- `clean_earthquake_data.py` - optional script that created the cleaned dataset.
- `Fuzzy_Logic_Earthquake_Report.pdf` - report PDF.

Run the manual program:

```powershell
python earthquake_fuzzy_logic.py
```

The program asks Input A until Input F, then prints the Mamdani and Sugeno output.
